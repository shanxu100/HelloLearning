## 接雨水
```java
/**
* https://leetcode-cn.com/problems/trapping-rain-water/
**/
public int maxArea(int[] height) {
    int i = 0;
    int j = height.length - 1;
    int maxResult = -1;
    while (i < j) {
        int area = Math.min(height[i], height[j]) * (j - i);
        maxResult = Math.max(maxResult, area);
        if (height[i] < height[j]) {
            i++;
        } else {
            j--;
        }
    }
    return maxResult;
}
```
